# 展柜组件演示

这是一个全新设计的展柜组件，用于展示你的项目、作品或任何精彩内容。

## ✨ 功能特点

- 🖼️ **图片完全填充** - 无留白，图片占据整个展示区域
- 📝 **标题在下方** - 清晰的标题显示在图片下方
- 💬 **悬停简短提示** - 鼠标悬停显示简短描述
- 🔍 **点击查看详情** - 点击图片弹出详细信息窗口
- 📱 **完美响应式** - 适配各种设备屏幕

## 使用方法

在你的 MDX 文件中导入组件：

```tsx
import Showcase from '@components/misc/Showcase.astro'
```

然后使用它：

```tsx
<Showcase 
  title="展柜标题"
  items={[
    {
      title: "项目名称",
      image: "图片URL或路径",
      description: "简短描述（悬停显示）",
      details: "详细描述（可选，点击后显示）"
    },
    // ... 更多项目
  ]}
/>
```

## 💡 使用提示

- **图片**：建议使用 4:3 比例的高质量图片
- **简短描述**：10-20 字，悬停时显示
- **详细描述**：100-300 字，点击后在模态窗口显示
- **支持本地图片**：可以使用 `/public` 目录下的图片
- **交互方式**：
  - 鼠标悬停 → 显示简短描述
  - 点击图片 → 打开详情窗口
  - ESC 键或点击背景 → 关闭窗口
